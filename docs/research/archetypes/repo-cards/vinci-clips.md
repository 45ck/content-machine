## Archetype

`longform-clip-factory`

## Production Pattern

Open-source web clipping platform that analyzes uploaded/link videos,
transcribes with speaker diarization, suggests optimal segments, and provides
dashboard review with transcript synchronization.

## Useful Extraction

- Gemini-powered video analysis
- speaker diarization and timestamp alignment
- smart clip generation
- upload progress and batch processing
- transcript-synchronized player
- dynamic reframing, AI captions, and audio enhancement claims

## Content-Machine Implication

Longform output should preserve transcript sync and speaker metadata through
candidate approval, crop planning, and caption render.
