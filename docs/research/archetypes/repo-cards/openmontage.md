# OpenMontage

Local path:
`vendor/imports-20260423-shortform-downloads-direct/direct-repos/calesthio__OpenMontage`

## Archetype

Agentic video studio; applies across `topic-faceless-explainer`,
`motion-graphics-lesson`, and reference-video workflows.

## Production Pattern

Agent selects a pipeline, reads manifests and stage skills, estimates cost,
generates or sources assets, renders in Remotion/HyperFrames, and runs
multi-point self-review.

## Useful Extraction

- pipeline selection before execution
- provider decision logs
- cost estimates before generation
- sample approval before full production
- real-footage path through licensed/open sources
- reference video as style/pacing input, not copied asset
- self-review gates with ffprobe/frame/audio/subtitle checks

## Content-Machine Implication

Content-machine should keep skill/flow surfaces as the decision layer and
promote provider/cost/review decisions into artifacts.
