# CLI Usage

Discover commands and options:

```bash
cm --help
cm <command> --help
```

Global flags:

- `--json` for schema-versioned machine output
- `--verbose` for debug logging
- `--offline` to disable on-demand downloads
- `--yes` to allow safe auto-downloads where supported

Pipeline commands:

- `cm generate`
- `cm script`
- `cm audio`
- `cm visuals`
- `cm render`

Utilities:

- `cm demo`
- `cm doctor`
- `cm setup`
- `cm research`
- `cm init`
