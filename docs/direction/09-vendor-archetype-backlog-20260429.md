# Vendor Archetype Backlog 20260429

This backlog records archetypes found in local vendor imports, research
exports, and Downloads CSV scans that are not yet proven as first-class
repo examples.

Keep `reddit-post-over-gameplay` as the flagship. Build the rest one at
a time, and do not promote a lane without a skill, request example, user
doc, demo MP4/GIF, and honest publish-prep status.

## Build Rule

For each lane:

1. Write the skill first.
2. Make one real demo.
3. Run publish-prep on the final MP4.
4. Document failures honestly.
5. Only then add it to README or user examples.
