# Animation Explainer Production Shape

Use this when the short needs to feel designed rather than merely
illustrated.

## Sequence

1. reduce the script to `3` to `6` visual teaching beats
2. decide each beat's primary mode:
   motion card, diagram, generated scene, icon board, or support clip
3. write shot prompts from structured scene intent, not one global style
   prefix
4. keep continuity references when generated scenes belong to the same
   world
5. verify scene variation before render
6. verify narration landmarks against visual landmarks before publish

## Good Defaults

- use typography for emphasis, not as the only visual system
- use generated scenes for impossible or metaphor-heavy beats
- use stock only where it clearly explains the spoken point
- cut every time the idea changes, not only when the sentence ends
