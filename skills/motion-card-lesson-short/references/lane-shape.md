# Motion Card Lesson Lane Shape

Use this for educational shorts that should feel like a designed card
sequence rather than a stock-footage explainer.

## Good Lane Shapes

- `explainer-card`
  hook -> claim -> three teaching beats -> proof -> recap
- `quiz-reveal`
  rules -> prompt -> countdown -> answer -> reinforcement -> CTA
- `flashcard`
  term -> clue -> reveal -> example -> recap

## Core Rule

Every card needs one job. If a card is trying to teach, decorate, and
caption the whole sentence at once, the lane is overloaded.
