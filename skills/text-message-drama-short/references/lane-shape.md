# Text Message Drama Lane Shape

Use this when the story should feel like a message reveal rather than a
Reddit post reading.

## Shape

1. Cold-open on the most suspicious or explosive bubble.
2. Reveal the exchange in short beats, not as one full-screen wall.
3. Zoom, crop, or restage the key messages as separate assets.
4. Use narrator glue between message beats.
5. End on the final text, block, confession, or reversal.

## Visual Priorities

- sender contrast first
- legibility second
- motion rhythm third

If you can only preserve one thing, preserve legibility.

## Two Good Reveal Modes

- `thread-log`
  Keep a persistent message stack visible and reveal turns inside it.
  Use when the running context matters.
- `floating-reveal`
  Show one bubble or receipt beat at a time with cleaner staging. Use
  when each line is a punch.

## Good Inputs

- breakup texts
- scam / wrong-number turns
- roommate / friend arguments
- parent / child drama
- “look what they sent me” reveal stories

## Bad Inputs

- giant unedited screenshot dumps
- messages that only make sense with lots of off-screen context
- stories where narration could carry everything better than the chat UI
