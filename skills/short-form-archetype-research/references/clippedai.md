# ClippedAI

Source: <https://clippedai.shaarav.xyz/>

## Archetype

`longform-clip-factory`

## Production Pattern

Open-source local alternative to OpusClip: input long video, detect
highlights, crop to 9:16, add animated subtitles, and generate titles.

## Useful Extraction

- local YouTube/local-file input
- highlight detection as an explicit stage
- face-tracking vertical crop
- word-by-word animated captions
- title and description generation attached to export

## Content-Machine Implication

Attach packaging metadata to selected clips earlier. A clip candidate should
carry hook/title/description candidates before render and publish review.

## Asset Policy

Use as architecture reference. Do not copy demo video assets.
