# Prompt

Create a 38-second listicle short explaining why split-screen Subway
Surfers videos with hard-cut captions are so difficult to ignore.

Requirements:

- Audience: people who know the meme but want the psychology
- Tone: sharp, explanatory, internet-native, not cringe
- Structure: hook inside the first 2 seconds, then 3 numbered reasons,
  then a tight payoff ending
- Each beat should be short enough to caption cleanly
- Avoid filler, generic motivation language, and vague claims
- Make the reasons concrete: attention reset, cognitive load, and
  constant novelty should all appear in plain English
