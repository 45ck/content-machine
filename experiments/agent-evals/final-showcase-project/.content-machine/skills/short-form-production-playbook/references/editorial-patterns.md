# Editorial Patterns

This is the short-form editorial layer aggregated from the imported
short-video repos plus this repo's own caption and validation stack.

## Content Shapes

- `explainer`
  Fastest path for tutorials, opinions, and tool demos.
  Shape: hook -> why it matters -> 2 to 4 beats -> payoff.
- `proof/result`
  Start with the result, then show why it worked.
  Use for before/after, speedups, and "I tried X" content.
- `reaction`
  Open with the moment worth reacting to, not the setup.
  Use commentary plus visual punch.
- `clip-and-comment`
  Use when the source material already carries the interest.
  Your job is segment selection, framing, captions, and payoff.
- `story`
  Setup -> turn -> payoff, but still keep every beat compact.

## Hook Rules

- Put useful visual information in frame 1.
- Spoken audio should start immediately unless silence is the hook.
- Text hooks work best when they complement the speech instead of
  restating it.
- Numbers, negations, contrarian claims, and concrete promises pull
  harder than generic topic introductions.
- If the first sentence could appear in a webinar intro, rewrite it.

## Script Density

- Around `35-40` words for a tight `15s` short.
- Around `70-80` words for `30s`.
- Around `125-150` words for `60s`.
- Remove throat-clearing, backstory, and duplicated claims before you
  touch visuals.

## Segment Selection

If you are clipping from long-form material, score candidates on:

- hook strength
- standalone coherence
- emotional intensity
- value density
- payoff quality

Reject segments that depend on missing context, fade in mid-thought, or
end without payoff.

## Pacing Rules

- Visual change every `1-3s` is a strong default.
- Hold longer only when the viewer is reading, inspecting, or waiting
  for a payoff.
- A short can be calm, but it cannot feel stalled.
- Use pattern interrupts on purpose, not as noise every second.

## Caption Relationship

- Caption design is part of the edit, not a post-process.
- Choose the caption family before picking dense visuals.
- If the short wants word-by-word aggression, simplify the frame.
- If the frame is already visually loud, lower caption motion before you
  lower legibility.

## Practical Prompt For The Agent

Before execution, write five lines:

1. Promise
2. Hook
3. Beat map
4. Caption family
5. Failure mode most likely to kill the short
