# Proving Prompt

Archetype: text-thread reveal

Target:

- produce one reviewable vertical MP4
- keep the visual identity as staged chat/message beats
- avoid generic stock footage as the primary grammar

Visual rules:

- cold-open on the most suspicious text bubble
- reveal the exchange in short beats, not a full unreadable wall
- keep sender contrast and legibility above decoration
- let captions support the narration without covering the key bubble

Runtime constraints:

- use repo-local harnesses where possible
- keep all lane-owned inputs and outputs under this directory
- report exact file paths and blockers if any stage fails
