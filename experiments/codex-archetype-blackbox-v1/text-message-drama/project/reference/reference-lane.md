# Text Message Drama Reference Lane

Target shape:

- top half is primarily message cards or receipt cards
- bottom half is continuous gameplay
- captions ride the midpoint overlay

Rules:

- first visible card should create immediate suspicion
- do not leave one message card static for too long; reveal or swap
  cards every `2s` to `4s`
- use actual message-card assets inside `assets/messages/`
- support video is optional, but the message lane remains the identity
- the ending should read like a reveal or “who was wrong” prompt
